Etiquetas seamanticas

Las etiquetas semánticas ayudan a definir la estructura del documento y permiten que las páginas web sean mejor indexadas por los buscadores. Una etiqueta se califica como semántica si tiene que ver el significado, es decir, si nos informa sobre lo que trata su contenido. La etiqueta

se usa para marcar un grupo de elementos de introducción o de navegación dentro de una sección o documento. La etiqueta
se usa para agrupar un conjunto de uno o más elementos de encabezado. La etiqueta
se usa para marcar una sección del documento cuya función es la navegación por la página web. La etiqueta
se usa para marcar contenido independiente que tendría sentido fuera del contexto de la página actual y que podría sindicarse, por ejemplo: una noticia, un artículo en un blog o un comentario. La etiqueta
se usa para marcar una sección genérica de un documento o aplicación. Una sección, en este contexto, es una agrupación temática del contenido, típicamente con un encabezado. La etiqueta
se usa para marcar un trozo de contenido que está relacionado con el contenido de la página web, pero que no es parte del mismo. La etiqueta
se usa para marcar el pie de una sección o documento y que contiene información sobre el mismo como el autor, licencia, términos de uso
